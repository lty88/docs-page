# 文档创建与组织

## 1. 文档结构规划

### 1.1 目录结构
```plaintext
docs/
├── guides/          # 用户指南
│   ├── basic/      # 基础指南
│   └── advanced/   # 高级指南
├── tutorial/        # 教程
└── reference/      # 参考文档
```

### 1.2 文档类型
- 教程（Tutorial）：适合新手入门
- 指南（Guide）：解决具体问题
- 参考（Reference）：详细的功能说明

## 2. 文件命名规范

### 2.1 基本规则
- 使用小写字母
- 用连字符(-)连接单词
- 名称要有描述性

### 2.2 示例
```plaintext
user-guide.md
getting-started.md
advanced-features.md
```
